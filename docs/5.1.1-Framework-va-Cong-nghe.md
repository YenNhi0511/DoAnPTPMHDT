# 5.1.1. Framework và Công Nghệ Sử Dụng

## 1. Kiến Trúc Hệ Thống

Hệ thống được xây dựng theo mô hình **3-tier architecture**:

- **Frontend**: React.js + TypeScript
- **Backend**: Django + Django REST Framework
- **Database**: PostgreSQL + Redis

## 2. Công Nghệ Frontend

| Công Nghệ            | Phiên Bản | Mục Đích                      |
| -------------------- | --------- | ----------------------------- |
| **React.js**         | 18.x      | Framework chính xây dựng SPA  |
| **TypeScript**       | 5.x       | Type safety, giảm lỗi runtime |
| **Redux Toolkit**    | 2.x       | Quản lý global state          |
| **Material-UI**      | 5.x       | Component library             |
| **React Hook Form**  | 7.x       | Form validation               |
| **Axios**            | 1.6+      | HTTP client                   |
| **Socket.IO Client** | 4.6+      | Real-time notifications       |

## 3. Công Nghệ Backend

| Công Nghệ                         | Phiên Bản | Mục Đích               |
| --------------------------------- | --------- | ---------------------- |
| **Django**                        | 5.0+      | Web framework          |
| **Django REST Framework**         | 3.14+     | RESTful API            |
| **djangorestframework-simplejwt** | 5.3+      | JWT authentication     |
| **Celery**                        | 5.3+      | Background task queue  |
| **Redis**                         | 7+        | Cache & message broker |

## 4. Database

| Công Nghệ      | Phiên Bản | Mục Đích                  |
| -------------- | --------- | ------------------------- |
| **PostgreSQL** | 15+       | Primary database          |
| **Redis**      | 7+        | Caching & session storage |

**Đặc điểm:**

- 9 bảng chính: users, jobs, applications, interviews, etc.
- UUID primary keys
- JSONB fields cho AI analysis data
- Full-text search indexes

## 7. Security

- **JWT Authentication**: Token-based auth
- **Django Security**: CSRF, XSS protection
- **HTTPS/SSL**: Encrypted communication
- **Environment Variables**: Sensitive data management

## 8. Kiến Trúc Triển Khai

```
Client (Browser)
       ↓ HTTPS
    Nginx (Load Balancer)
       ↓
   ┌───┴────┬─────────┐
   ↓        ↓         ↓
React    Django    Celery
         ↓         ↓
   PostgreSQL   Redis
```

## 9. Lý Do Chọn Stack

**Django + React:**

- Tách biệt Frontend/Backend rõ ràng
- Python ecosystem mạnh cho AI/ML
- React component-based, dễ tái sử dụng

**PostgreSQL:**

- JSONB support cho AI analysis data
- Full-text search built-in
- ACID compliance

**Redis:**

- Cache queries thường xuyên
- Message broker cho Celery
- Session storage

## 10. Kết Luận

Stack công nghệ được chọn đảm bảo:

- **Performance**: Fast, responsive, scalable
- **Security**: Industry-standard practices
- **Maintainability**: Clean architecture
- **AI Integration**: Python ecosystem mạnh
- **Cost-effective**: Open-source technologies
